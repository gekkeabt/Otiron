Otiron
======

Otiron is a self-hosted scrum board/trello alternative.

Installation
=====
1. Place all the files in a folder
2. Browse to the folder in cmd/node command line
3. Type `npm install`
4. After done type `node app`
5. Browse to `http://127.0.0.1:4999`
6. Enjoy!
